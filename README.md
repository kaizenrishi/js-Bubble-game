# Bubble Game

Play it live: [js-Bubble-game on Vercel](https://js-bubble-game-jade.vercel.app)

A simple and interactive web game built with JavaScript, HTML and CSS.

## Gameplay
- Click the bubble with the target number as quickly as possible to earn points!
- The number and position of bubbles keep changing, testing your speed and accuracy.
- Points are awarded for correct clicks; incorrect clicks may subtract points or end the game.

## Features
- Clean interface with user-friendly design
- Real-time score tracking
- Timer for added challenge
- Responsive layout for desktop and mobile

## How to Play
1. Run the project locally or open the live demo link above.

1. Clone this repository:

2. Open `index.html` in your browser. No build step or dependencies required.
